# Результаты по прогонам

Пока расклад такой:

**QWEN и FireRed** — сейчас практически наилучшая связка, и они хорошо дополняют друг друга. FireRed — эталон по смене времени суток (как раз там, где QWEN слабовата), зато QWEN лучше всех переодевает людей и аккуратнее всего сохраняет остальное (например, не стал убирать туман на "туманном" фото — и это как раз плюс: часть других моделей туман убирает и дорисовывает какую-то отсебятину). Наихудшие результаты показали **Flux2-dev** и **Flux1-kontext-dev**. **longcat-image-edit** часто не понимает геометрию помещений и ландшафта.

## Как читаем оценки

Каждая задача оценивается по рубрике `quality / artifacts / adherence / preservation` (1–5).
**Preservation (P) — приоритет**: меняет ли модель то, что менять не просили.

В таблице ниже — **средний P по 5 задачам** для каждого прогона.

## Коротко по прогонам

- **Прогон 1** — Модели на SD-токенах, seed ещё не фиксировался. Уже видно лидеров (QWEN, FireRed) и аутсайдера (klein с P=1.8).
- **Прогон 2** — Почти все вытянули на 4+. Тогда еще не выработал точную методику сравнения.
- **Прогон 3** — перешли на финальный сложный набор + переодевание в клоунов. Тут просели flux1/flux2-dev/longcat: костёр вместо возгорания, ломаная геометрия, не всех людей одевают в костюм.
- **Прогон 4** — правки промптов (strength вниз, утвердительные формулировки). QWEN стабильно держит P≈4.4, FireRed ровный, остальные без скачка.
- **Прогон 5** — фикс **klein → cfg=1** (была ошибка воркфлоу): preservation 2.0 → 3.2. longcat подрос за счёт шагов 50 и понижения FluxGuidance 4.5 → 3.0.

## Итоговый вывод по моделям

| Модель | Вердикт |
|---|---|
| **qwen-image-edit-2511** | Лучшая по сумме и лучшая в переодевании людей. Слабое место — смена времени суток (там средне). Главный рабочий вариант. |
| **firered-image-edit-1.1** | Уверенное второе место. Эталон по смене времени суток. Иногда десатурирует картинку. |
| **longcat-image-edit** | Хороша на deblur/normalize, но плывёт на геометрии сцены (возгорание, ландшафт). |
| **flux2-klein-9b** | После фикса cfg=1 выбралась из дна, но всё ещё нестабильна по preservation. |
| **flux1-kontext-dev** | Слабая на сложном наборе: плохо с возгоранием и переодеванием. На финал — не прошла. |
| **flux2-dev** | Аутсайдер. Q4_K_M-квант пересвечивает цвета, preservation стабильно низкий. На финал — эксперимент с SD-токенами. |

> На финальный прогон 5 моделей(без flux1). flux2-dev переведен на SD-токен-стиль как эксперимент — посмотреть, вытянет ли это самые слабые модели. Seed зафиксирован (`FIXED_SEED=12345`), так что сравнение с instruction-версией будет чистым.
